# Execução:
- Abrir o terminal no diretório do projeto e executar os seguintes comandos:<br/><br/>
  Para instalar a pasta node_modules:

 ```sh
 npm install
 ```
 <br/>
 Para startar a aplicação:

  ```sh
 npm start 
 ```
 Para fazer o build:

 ```sh
 npm run build
 ```
